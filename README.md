# ShopHere 

> MERN Stack Ecommerce Project using React, Redux, Node.js, Express, MongoDB, React_Bootstrap
>


### Install Dependencies (Frontend)

```
cd client
npm i
```
### Run Frontend

```
cd client
npm start
```

### Install Dependencies (Backend)

```
cd backend
npm i
```

### Run Backend

```
cd backend
npm start
```
